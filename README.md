# fraud-detection-system
• Built a fraud detection platform using XGBoost and Scikit-learn to automate identification of high-risk transactions and reduce reliance on manual review processes.

• Engineered predictive features and anomaly detection workflows to uncover hidden fraud patterns within large transactional datasets.

• Optimized and validated models using ROC-AUC, Precision, Recall, and F1-Score metrics to balance fraud detection effectiveness and false-positive rates.
